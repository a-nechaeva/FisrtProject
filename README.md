# 🌃Домашнее задание 2

* создано приложение с двумя Activity;
* на экран первой Activity добавлены поле EditText для ввода текста и кнопка "Открыть вторую Activity", при нажатии на которую отображается вторая Activity с текстом из EditText;
* на экран первой Activity также добавлена кнопка "Позвонить другу", при нажатии на которую используется неявный Intent для открытия приложения звонков (ACTION_DIAL) с номером телефона из поля EditText;
* на экран первой Activity добавлена кнопка "Поделиться текстом", при нажатии отправляется Intent (ACTION_SEND) с текстом из EditText и появляетсяся системное окно "Поделиться через…".
* В качестве валидных данных принимается непустая строка или корректный номер телефона (может начинаться с + или с цифры и состоять из 7-15 цифр.
